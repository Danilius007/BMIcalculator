# Калькулятор ИМТ

Приложение для расчёта **индекса массы тела (ИМТ)** с визуальным отображением результата, историей вычислений и поддержкой метрической и имперской систем единиц.

---

## Возможности

- Расчёт ИМТ по стандартной формуле
-  Поддержка метрической и имперской систем измерения
-  Визуальное отображение категории веса
-  История вычислений с сохранением в SQLite
-  Горячие клавиши:
  - `Ctrl + H` — открыть историю
  - `Ctrl + Q` — выйти из приложения

##  Запуск проекта

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/Danilius007/BMIcalculator.git
cd BMIcalculator
```

### 2. Создайте виртуальное окружение

```bash
python -m venv venv
```

### 3. Активируйте виртуальное окружение

```bash
venv\Scripts\activate
```

### 4. Установите зависимости

```bash
pip install -r requirements.txt
```

### 5. Запустите приложение

```bash
python main.py
```

---

##  Формула расчёта

```text
ИМТ = вес (кг) / рост² (м²)
```

Для имперской системы используется эквивалентная формула с автоматическим переводом единиц.
